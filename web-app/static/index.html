Privately Promoted

Welcome to the Privately Promoted. Please chose which stakeholder you want to see the viewpoint from.

User Dashboard

This stakeholder view represents internet users and how they interact with our marketplace. It includes the insights that social medias and other insight generators have created. Here users should select which insights are used for targeted advertising. It also includes a gallery view of advertisements their filtered insights allow them to see, their associated justification (what insights were used), and their associated revenue.

Advertiser Dashboard

This stakeholder view represents the interactions advertisers would have with our platform. They can create advertisements based on certain preferences, and how competitively they would like to bid for their ads to be selected. It includes the cost incurred by displaying each ad and other relevant details.

Backend Dashboard

This view shows the data flows at every step of the adselection. It start off with the total unfiltered insights sent by social medias to users. This then shows which insights were selected by users. Then a website "generates" traffic and connects a user with an ad. Several ads are considered for the selection, but most fail to match the requirements of the users and the website. Finally an advert is selected

Insight Generator

In this view an insight generator can view and create insights for users (randomly). Simply enter a user in the textbox and click "Generate Insihgts". And from the backend, all the relevant files are created which can be seen as buttons once the page has been reloaded.

Example Website

This view allows websites to select what sorts of advertisements are displayed on their website. They can also choose to "Create Traffic" simulating a user browsing their website, and thus initiates an ad seletion.
